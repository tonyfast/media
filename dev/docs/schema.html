qshop schema¶

Notes¶

it configures:

  • sphinx

  • latex

  • nikola

    both nikola and sphinx use the conf.py convention. nikola using capital letters for variables names while sphinx uses lowercase. because of these differences we can configure documentation for both in the same file, and share configuration metadata.

Note¶

the descriptions were taken from the jb docs https://jupyterbook.org/customize/config.html
